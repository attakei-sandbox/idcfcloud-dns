=============
idcfcloud-dns
=============

IDCF cloud DNS client for python


Installation
============


.. code-block:: bash

   $ pip install git@gitlab.com:attakei/idcfcloud-dns.git


Usage
=====

Pre
---

You must create IDCF-cloud account.

Command-line
------------

.. code-block:: bash

   $ export IDCF_DNS_API_KEY=your_api_key
   $ export IDCFDNS_SECRET_KEY=your_secret_key
   $ idcfcloud-dns list
