=============
idcfcloud-dns
=============

IDCF cloud DNS client for python


Installation
============


.. code-block:: bash

   $ pip install git@github.com:attakei/idcfcloud-dns.git


Usage
=====

Pre
---

You must create IDCF-cloud account.

Command-line
------------

.. code-block:: bash

   $ idcfcloud-dns list
